1) Working with Help Files

?mean
Note: this works without loading anything because it is in base R
  • Now, we can look through the help file. The format is always the same. At the top left-hand side, it gives you the function name with the associated package in brackets. Here, we see mean{base}
  • The description gives context about what the function does
  • The usage tells you the default arguments and defines the arguments
    • Here, the first argument is x (an R object), trim = 0, and na.rm = F)
    • This can be a little confusing to read through, but keep in mind, you want to know what your defaults are
  • Take a look at na.rm, unless you specifically tell R to remove missing values, it will give you a mean of NA if you have a vector with missing values
  • Lastly, looks through the examples

Step 1: Google

If you are working on something in R, and you have no idea where to start, it’s best to ask Google. (Then why did I spend all that time with help files? You will see!)

  • Practice: How can we rename multiple variables in this dataset?
  • Tips: Try using precise, technical language
    • Instead of “change the name of a variable in R” try “rename columns in R”
    • Instead of “how to make the writing in the center on rmd” try “center alignment/justify text rmd”
Google how to do this and copy/paste example code from online or a function that could accomplish this task in the chat!

Part III: Common Errors

  1. Function not found
  2. Object not found
  3. Non-numeric argument to a binary operator
  4. File not found
  5. RS (It’s like ‘BS’ but for R)

1) Function not found

Example 1:

mtcars %>% 
  remane(weight = wt)
## Error in remane(., weight = wt): could not find function "remane"

Example 2: 

principal(mtcars, nfactors = 2)
## Error in principal(mtcars, nfactors = 2): could not find function "principal"

  • Lessons:
    • Check spelling
    • Load libraries

In the first example, we see that I made a spelling mistake. Easy fix!

In the second example, I did not make a spelling error, but it still does not work?? Why! Because I did not load the library(psych) which calls to principal; Remember: check spelling, and check libraries!

2) Object not found

mtcars %>%  
  rename(weight = tw)
## Error: Can't rename columns that don't exist.
## [31mx[39m Column `tw` doesn't exist.
  • Lesson:
    • Check spelling
    • Check your codebook

3) Non-numeric argument to a binary operator

mtcars %>% 
  rownames_to_column(var = "make_model") %>% 
  mutate(make_model = make_model + 1)
## Error: Problem with `mutate()` input `make_model`.
## x non-numeric argument to binary operator
## i Input `make_model` is `make_model + 1`.
  • Lessons:
    • Check your codebook
    • Know your data structure

4) File does not exist

practice_dat <- read_csv("practice_ds.csv")
## Error: 'practice_ds.csv' does not exist in current working directory ('C:/Users/Amanda/Google Drive/Work/Teaching').
  • Lessons:
    • Check your working directory getwd() and change it if necessary setwd("path")
    • Know where your files are

5) RS (It’s like ‘BS’ but for R)

Sometimes, you think you’ve done everything right, and it still won’t work.

  • In this case, check to see if package is installed
  • If yes, then, turn it off and turn it on again
    • I’m serious…
    • Hence “R.S.”
